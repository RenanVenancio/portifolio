Digi - Catálogo Digital

Trata-se de uma plataforma de cardápio/catálogo digital, onde as empresas podem compartilhar o link da sua loja em uma mídia social, como WhatsApp, Instagram, etc, e seus clientes podem acessar seus catálogos através do link e realizar os pedidos on-line. Foi desenvolvida também a área administrativa, onde a empresa pode consultar seus pedidos, cadastrar produtos, categorias, e visualizar alguns relatórios. Toda a comunicação da plataforma é feita através de uma API Rest, desenvolvida com spring, toda a documentação foi feita com Swagger, a camada de segurança da aplicação utiliza tokens JWT. Já no front-end utilizei o React, hooks e context api para gerenciar os estados globais, a estilização deixei por conta do rainbow-ui.

Finder - Plataforma para busca de pessoas desaparecidas

Nos últimos meses estive realizando estudos sobre reconhecimento e detecção facial para desenvolvimento do meu trabalho de conclusão de curso, decidi me aventurar no mundo da visão computacional utilizando a biblioteca OpenCv e Python, os estudos envolveram também outras tecnologias como React, Django, o próprio Opencv e os algoritmos de machine learning Haar Cascade e LBPH. O resultado final foi o desenvolvimento de uma plataforma Web onde pode-se realizar o cadastro de pessoas desaparecidas e utilizar-se do reconhecimento facial para identificar essas pessoas, e notificar familiares via e-mail caso as mesmas sejam encontradas.
